Class: Map

Map

ol.Map 是openlayers的核心地图组件. 为了渲染地图, a view,
一个后者多个图层, 一个DOM容器 是必须需要的:

var map = new ol.Map({
  view: new ol.View({
    center: [0, 0],
    zoom: 1
  }),
  layers: [
    new ol.layer.Tile({
      source: new ol.source.OSM()
    })
  ],
  target: 'map'
});

上面的代码片段用一个 map 通过 ol.layer.Tile 来展示
ol.source.OSM OSM 数据 然后通过DOM id为 ‘map’ 的元素来渲染地图
